# Variables and Constants in JavaScript

This session covers:
- `var`: Function-scoped, hoisted
- `let`: Block-scoped, not hoisted
- `const`: Block-scoped, immutable
- Scope: Global vs Local
- Hoisting: Declarations vs Initializations

## Examples included:
- `01-var.js`
- `02-let.js`
- `03-const.js`
- `04-scope.js`
- `05-hoisting.js`
